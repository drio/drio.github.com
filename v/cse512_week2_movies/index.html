The Movie Dataset

This is an exploratory data analysis exercise that uses this movies dataset.

Inspecting the data

The csv file contains 3201 movies. For each movie we have the following attributes (not all the movies have all the attributes set): 

0: "Title" 1: "US Gross" 2: "Worldwide Gross" 3: "US DVD Sales" 4: "Production Budget"
5: "Release Date" 6: "MPAA Rating" 7: "Running Time (min)" 8: "Distributor"
9: "Source" 10: "Major Genre" 11: "Creative Type" 12: "Director"
13: "Rotten Tomatoes Rating" 14: "IMDB Rating" 15: "IMDB Votes"

Let's inspect the number of uniq values available per attribute on the dataset:
