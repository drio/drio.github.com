This is a visualization for the dataset from this make over monday challenge. It is data related to the amount of power generated using wind technology by the different US states.

I added some extra data to see how the population and area of a state change with the other variables provided in the challenge.

I find the parallel plot very effective to explain this dataset. Thank you to the developers of d3.parcoords.js. Thanks to them, I didn't have to code this visualization from scratch. You guys did a great job with the API also.

This viz was also a good opportunity to explor brushing and zooming. Two interaction techniques I believe work well in this context. Feel free to zoom in the dotplots and brush along the axis in the parallel plot.
