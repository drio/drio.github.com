For this project, the customer had a set of dashboards created using microsoft's powerBI tool. They use the dashboards to communicate the results from different analytical pipelines. The project involved two main aspects: first, building a system to automatically consume data from those pipeline and two, redesign the different dashboards to communicate the data more effectively.
